# Product Steering

## Purpose

じゃんけんゲームとDevBoardを含むNext.jsアプリケーション。学習プロジェクトとして、ゲーム機能とGitHub統合機能を提供する。

## Core Value

- **ゲーム体験**: インタラクティブなじゃんけんゲームで、アニメーションと効果音による楽しい体験を提供
- **開発可視化**: DevBoardを通じてGitHubのコミット、Issue、Pull Requestを統合表示し、開発活動を可視化

## Key Capabilities

### じゃんけんゲーム
- ユーザーとコンピューターの対戦
- リアルタイムスコア追跡
- アニメーションとWeb Audio APIによる効果音
- レスポンシブデザイン

### DevBoard
- GitHub API統合（コミット、Issue、Pull Request）
- アクティビティの時系列表示
- カンバンビュー（準備中）

## User Experience Principles

- **日本語UI**: すべてのユーザー向けテキストは日本語
- **直感的なナビゲーション**: 明確なリンクとナビゲーション構造
- **視覚的フィードバック**: アニメーションと色による状態表示
- **エラーハンドリング**: ユーザーフレンドリーなエラーメッセージ
